今回のサイトはスポカレのLPです。

学校で勉強したReactを使いサイトを実装したいと思い、今回のサイトではReactと使い実装しました。

ファーストビューや機能説明など大まかに９個のコンポーネントに分けております。

そして、テキストデータを全部一つのファイルの書いていて、
そこから、コンポーネントごとに必要になった際に取り出したり、
今後テキストの変更があった際には全部一箇所に変更できるようにしました。
